# QGIS 설치 및 CN값 계산기 플러그인 설치

## 1. QGIS 설치
- 다운로드: https://qgis.org/download/
- 버전: **QGIS 3.34 LTR** (장기 지원 버전) 권장, 최소 3.0 이상
- 설치 시 별도 옵션 변경 없이 기본값으로 설치

## 2. Python 라이브러리 설치
시작 메뉴 → **OSGeo4W Shell** → 관리자 권한으로 실행 후:
```
pip install psycopg2-binary pandas openpyxl
```

## 3. 플러그인 설치
1. QGIS 실행
2. 메뉴: **플러그인** → **플러그인 관리 및 설치** → **ZIP에서 설치**
3. 전달받은 **`gis_cn.zip`** 선택 → **플러그인 설치**
4. **설치됨** 탭에서 **"GIS CN값 계산기"** 체크박스 활성화
5. 툴바에 CN 아이콘이 나타나면 완료
